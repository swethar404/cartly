# 🛒 Cartly

A Flutter-based smart retail management application designed to simplify retail operations through an intuitive and user-friendly mobile interface.

---

## 📌 Overview

Cartly is a cross-platform mobile application developed using Flutter and Dart. The application focuses on providing an efficient and responsive solution for retail management while delivering a clean user experience.

---

## ✨ Features

- User Authentication
- Responsive User Interface
- Product Management
- Intuitive Navigation
- Cross-platform Mobile Application
- Scalable Application Structure

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/swethar404/cartly.git
```

### Navigate to the Project

```bash
cd cartly
```

### Install Dependencies

```bash
flutter pub get
```

### Run the Application

```bash
flutter run
```
